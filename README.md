## ✨ What's New in v6.0

### Calc Engine ครบวงจร
- 🏭 **5 ประเภทอาคาร**: โกดัง, โชว์รูม, โรงงาน, อาคาร 1 ชั้น, 2 ชั้น
- ═ **แป Purlin** — C-channel เลือกหน้าตัดอัตโนมัติ
- ║ **อกไก่ Girt** — แปผนังรับลม
- ✕ **ค้ำยัน Bracing** — เหล็กฉากรับแรงลม
- 🏗️ **คานเครน** — 3-50 ตัน
- ▦ **ชั้นลอย Mezzanine** — คำนวณคาน/พื้นที่
- 🌪️ **แรงลม มยผ.1311** — Cp ทุกด้าน, Base Shear, Uplift
- 🌧️ **ระบบระบายน้ำ** — รางน้ำ, ท่อ, บ่อพัก
- ⚡ **MEP เบื้องต้น** — ไฟฟ้า, ประปา, ดับเพลิง
- 🧱 **วัสดุเลือกได้** — ผนัง 5 ชนิด, หลังคา 4 ชนิด, ดิน 4 ชั้น, ลม 3 โซน
